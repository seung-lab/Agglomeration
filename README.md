## Installation
```
- Pkg.clone("git@github.com:seung-lab/Agglomeration.git")
- Pkg.build("Agglomeration")
```
The Vis module requires PyQtGraph.

- `using Agglomeration`
- `using Process`
- `dend, dendValues = forward(aff, segm)`
